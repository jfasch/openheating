Components
==========

OpenHeating "components" are `D-Bus
<https://www.freedesktop.org/wiki/Software/dbus/>`__ nodes. Each node
is a standalone excutable which attaches to D-Bus, acquiring a *bus
name* and exposing one or more *objects*. Objects provide
*interfaces*.
